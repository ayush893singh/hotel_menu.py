## Restaurant Billing System
A simple Python program that allows users to order food items and calculates the total bill.

## Features
* Menu-based ordering system
* Add multiple items
* Calculates total bill
* User-friendly interface

##  Technology Used
* Python
* Dictionary
* Loops & Conditions

## How to Run

1. Install Python
2. Download this repository
3. Run:
   python restaurant.py
